## Official Resources

- [Next.js Docs](https://nextjs.org/docs) – Official documentation.
- [Next.js GitHub](https://github.com/vercel/next.js) – Source code and development.
- [Next.js Examples](https://github.com/vercel/next.js/tree/canary/examples) – Full list of example apps.
- [Vercel](https://vercel.com/) – Hosting platform made for Next.js.

## Starter Kits & Templates

- [Vercel Templates](https://vercel.com/templates/next.js) – Official templates maintained by Vercel.
- [Next.js Boilerplate](https://github.com/vercel/nextjs-boilerplate) – A clean, modern baseline.
- [shadcn/ui Next Template](https://github.com/shadcn/ui/tree/main/apps/www) – Starter with shadcn/ui.
- [Next.js SaaS Starter](https://github.com/async-labs/saas) – SaaS boilerplate with authentication, billing, and UI.
- [T3 Stack](https://github.com/t3-oss/create-t3-app) – TypeScript-first full-stack starter (Next.js + tRPC + Prisma + Tailwind).
- [Next.js Starter Kit (Supabase)](https://github.com/supabase/supabase/tree/master/examples/nextjs) – Full-stack starter with Supabase backend.
- [Next Boilerplate](https://github.com/pankod/next-boilerplate) – Production-ready boilerplate.

## UI Libraries & Components

- [Tailwind CSS + Next.js](https://tailwindcss.com/docs/guides/nextjs) – Official integration guide.
- [shadcn/ui](https://github.com/shadcn/ui) – Popular component library built for Next.js + React.
- [Mantine](https://github.com/mantinedev/mantine) – React component library with SSR support.
- [Material UI](https://github.com/mui/material-ui) – Well-known React UI library with server-side rendering compatibility.
- [Radix UI](https://github.com/radix-ui/primitives) – Unstyled, accessible UI primitives.
- [NextUI](https://github.com/nextui-org/nextui) – Beautiful React UI built for Next.js.
- [Chakra UI](https://github.com/chakra-ui/chakra-ui) – Simple, accessible component library.

## Full-Stack & Backend Tools

- [tRPC](https://github.com/trpc/trpc) – End-to-end typesafe APIs for Next.js.
- [Hono](https://github.com/honojs/hono) – Ultra-fast web framework usable inside Next.js routes.
- [NestJS + Next.js](https://github.com/nestjs/nest) – Typical pair for enterprise full-stack apps.
- [OpenAPI TypeScript Codegen](https://github.com/ferdikoomen/openapi-typescript-codegen) – Generate client/server types for API routes.
- [Zod](https://github.com/colinhacks/zod) – Schema validation used extensively with Next.js.

## State Management

- [Zustand](https://github.com/pmndrs/zustand) – Minimal, scalable store.
- [Jotai](https://github.com/jotai-labs/jotai) – Primitive atom-based state management.
- [Recoil](https://github.com/facebookexperimental/Recoil) – Facebook’s atom-based global state.
- [Redux Toolkit](https://github.com/reduxjs/redux-toolkit) – Modern Redux for React/Next.js.

## Authentication

- [NextAuth.js](https://github.com/nextauthjs/next-auth) – The most popular authentication library for Next.js.
- [Clerk](https://github.com/clerkinc/clerk-nextjs) – Full-stack auth, SSO, MFA, user management.
- [Auth0 Next.js SDK](https://github.com/auth0/nextjs-auth0) – Auth0 integration for Next.js apps.
- [Supabase Auth](https://supabase.com/docs) – Open-source Firebase alternative with Next.js examples.

## Databases & ORM

- [Prisma](https://github.com/prisma/prisma) – Type-safe database ORM.
- [Drizzle ORM](https://github.com/drizzle-team/drizzle-orm) – Lightweight SQL ORM with schema-first TypeScript.
- [PlanetScale](https://planetscale.com/) – MySQL serverless platform with Next.js guides.
- [Supabase](https://supabase.com/) – Postgres backend with Next.js examples.
- [Neon Postgres](https://neon.tech/) – Serverless Postgres with edge capabilities.
- [Turso + libSQL](https://turso.tech/) – Distributed SQLite for edge Next.js deployments.

## Deployment

- **Vercel** – The official Next.js hosting platform.
- [Netlify](https://www.netlify.com/) – Supports Next.js functions and SSR.
- [Cloudflare Pages](https://developers.cloudflare.com/pages/framework-guides/nextjs/) – Fast edge hosting for Next.js (experimental SSR).
- [AWS Amplify](https://docs.amplify.aws/) – Next.js support for full-stack apps.
- [Fly.io](https://fly.io/) – Deploy globally on micro-VMs.

## Performance & Monitoring

- [Vercel Analytics](https://vercel.com/analytics) – Built-in performance and traffic analytics.
- [OpenTelemetry](https://opentelemetry.io/) – Instrumentation for distributed tracing.
- [Sentry](https://github.com/getsentry/sentry-javascript) – Error and performance monitoring.
- [LogRocket](https://logrocket.com/) – Session replay and tracking.
- [Speed Insights](https://vercel.com/speed-insights) – Real user monitoring (RUM).
